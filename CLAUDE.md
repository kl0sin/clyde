# Clyde — repo conventions for Claude Code

## Commits & attribution

- **Never** add `🤖 Generated with Claude Code`, `Generated with Claude`,
  or any equivalent footer to commit messages.
- **Never** add `Co-Authored-By: Claude …` (or any Claude/Anthropic
  identity) to commits.
- **Never** insert generated-by markers into source files, docs,
  comments, README, CHANGELOG, or any other artifact.

Commits should look exactly as if a human wrote them. Author and
co-author fields stay clean.
