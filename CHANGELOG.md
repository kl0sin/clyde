## [0.1.0] — 2026-04-07

First public release.

### Highlights

- Real-time Claude Code session tracking via native hooks (no polling).
- Floating menu bar widget showing dominant state at a glance:
  ready · working · needs input.
- Expanded view with multi-session list, drag-to-reorder, custom names.
- Activity timeline of recent prompts, permissions and session lifecycle.
- Sound + macOS notification on attention requests.
- Snooze (15m / 30m / 1h / 2h) for quiet hours.
- Global hotkey (⌃⌘C) to expand from anywhere.
- Auto-installs and self-heals the Claude Code hook script.
- Universal binary, code-signed and notarized for macOS 13+.
- Auto-updates via Sparkle.
